Kawan_Art — Site (React + Vite + Tailwind)
=========================================

Conteúdo:
- Projeto React com Vite e Tailwind.
- Calendly embed configurado para: https://calendly.com/kawan_art/tatuagem
- Imagens de exemplo em /public/images (substitua pelas suas imagens reais).

Como usar localmente:
1. Instale dependências:
   npm install

2. Rode em desenvolvimento:
   npm run dev

3. Build para produção:
   npm run build
   npm run preview

Deploy na Vercel (passo a passo):
1. Suba este projeto para um repositório no GitHub.
2. Crie uma conta em https://vercel.com e conecte o GitHub.
3. Em Vercel, clique em "Import Project" > selecione seu repositório.
4. Build command: npm run build
   Output directory: dist
5. Clique em Deploy. O site estará disponível em segundos em um domínio vercel.app.

Substitua:
- Arquivos em /public/images com suas fotos reais (tatuagens, retrato do artista e fundo).
- O link Calendly em src/App.jsx se desejar outro.

Se quiser, eu mesmo faço o deploy na Vercel para você — preciso apenas de acesso ao repositório do GitHub (ou você pode me dar permissão temporária).
